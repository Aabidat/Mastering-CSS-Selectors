# CSS Selectors – Exercise Set

This is a set of six exercises. They help you learn how CSS selectors work and how to use them to style elements on a page.


## Exercise 1 – The Specificity Showdown

You are given a paragraph that has an ID and a class. Three CSS rules give it three different colors. You have to find out which rule wins and why. Then you have to make the paragraph green by adding just one new rule.

**You will practice:** selector weight, overriding rules, and combining selectors.


## Exercise 2 – The Grouping and Nesting Debug

Your teammate wrote CSS that has three mistakes. Links on the whole page are losing their underline. A heading rule does nothing. A list style is going where it should not. You have to find and fix all three.

**You will practice:** descendant selectors, group selectors, and the child combinator.


## Exercise 3 – Target Practice Logic Puzzle

You are given an HTML structure. You cannot change the HTML. You have to write one CSS rule for each target. You will color a list item, give a paragraph a background, and set a line height on both paragraphs. There is also a challenge question about why one selector does not work on this structure.

**You will practice:** nth-child, nth-of-type, last-child, and descendant selectors.


## Exercise 4 – Interactive Profile Card

You are given HTML for a profile card. You have to write all the CSS in an external file. The card has a heading, a paragraph, a button, and an input field. The button needs a hover effect. The input needs a focus effect. At the end, you use an ID to remove a margin that a group selector added.

**You will practice:** group selectors, hover, focus, transform, and ID specificity.

## Exercise 5 – Predict the Output

There are two scenarios. In each one, you read the CSS and HTML and predict what color the text will be. Then you explain why before you test it.

**You will practice:** inheritance, direct targeting, and specificity.


## Exercise 6 – Reconstruct from Scratch

No starter code is given. You build a Tech News sidebar from a blank file. You write the HTML and the CSS yourself based on a description. The CSS must use many different selector types from the exercises.

**You will practice:** everything - universal, element, ID, class, group, descendant, child, and pseudo-class selectors all in one project.



## Selectors Covered

- Universal selector `*`
- Element selectors 
- Class selectors
- ID selectors
- Group selectors
- Descendant selectors
- Child combinator selectors
- Pseudo-class selectors — `:hover`, `:focus`, `:nth-child`, `:nth-of-type`, `:first-child`, `:last-child`
- Specificity and inheritance
